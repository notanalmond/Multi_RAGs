# Multi_RAGs Streamlit Application

## Features
- **Data Loading**: Compatible with various data formats, enhancing flexibility in data handling.
- **Interactive Widgets**: Sliders, dropdowns, and buttons allow for real-time data manipulation.
- **Visualizations**: Employs libraries like Matplotlib and Seaborn to produce insightful charts and graphs.

## Prerequisites
Ensure the following are installed:
- Python (3.6+ recommended)
- Streamlit

## Installation
Clone the repository:
git clone https://github.com/notanalmond/Multi_RAGs/.git

## Usage
To run the application, navigate to the directory containing app.py and execute:
streamlit run app.py
